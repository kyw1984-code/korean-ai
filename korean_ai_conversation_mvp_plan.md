# 한국어 AI 회화 연습 앱 — MVP 기획안

> **한 줄 소개**: 유럽 한국어 학습자를 위한 AI 회화 파트너. 광고 시청으로 AI 비용을 상쇄하는 지속가능한 수익 모델.
> **공식 앱명**: KoreanTalk: AI Conversation Practice

---

## 1. 전략적 배경

### 1.1 기존 앱의 자산

현재 운영 중인 단어/어휘 학습 앱이 유럽 유저 기반에서 좋은 반응을 얻고 있다. 이는 다음을 의미한다.

- **검증된 타겟 시장 확보** — 신규 유저 획득 비용(CAC) 거의 0
- **ASO 자산 축적** — 한국어 교육 관련 키워드 상위 노출 경험
- **유저 피드백 축적** — 유럽 학습자의 진짜 페인 포인트 파악

### 1.2 왜 지금 AI 회화인가

단어를 외운 학습자의 다음 니즈는 **"실제로 말해보기"**다. 이 단계에서 대부분의 학습자가 좌절하고 이탈한다.

- Duolingo 한국어 → 규칙 기반, 자유 회화 불가
- Talk To Me In Korean → 강의 중심, 쌍방향 회화 없음
- iTalki/Preply → 원어민 튜터, 시간당 $10~25로 부담
- **AI 회화 앱 → 24시간 부담 없이, 저렴하게, 무한 반복 연습 가능**

이 갭을 메우는 게 핵심 기회.

### 1.3 유럽 유저 타겟의 강점

- **고ARPU 시장** — 독일·프랑스·영국 유저의 광고 eCPM은 동남아의 3~5배
- **K-콘텐츠 관심도 최상** — K-드라마/K-팝 통한 한국어 유입 급증
- **프라이버시/품질 민감** — GDPR 친화적 설계가 곧 차별화
- **구독 문화 정착** — 월 구독 $4.99~9.99 저항 낮음

## 2. 제품 포지셔닝

### 2.1 한 문장 포지셔닝

"Duolingo로 단어를 외웠다면, 이제 KoreanTalk으로 말해보세요. 하루 15분, AI 선생님과 실제 대화로."

### 2.2 핵심 차별화 3가지

1. **상황별 롤플레이 회화** — "카페 주문", "택시 타기", "친구 만나기" 등 실생활 시나리오 100개+
2. **실시간 교정 + 한국어식 사고 교정** — 단순 문법뿐 아니라 자연스러운 한국어 표현으로 바꿔줌
3. **광고 시청으로 무료 사용 가능** — 돈 없어도 꾸준히 쓸 수 있는 구조

### 2.3 기존 앱과의 연계

- 기존 단어 앱에서 학습한 단어를 AI 회화에 자동 반영 ("이 단어 써서 대화해봐")
- 크로스 프로모션: 기존 앱 배너 → 신규 앱 다운로드
- 유료 유저는 양쪽 앱 번들 구독 옵션 ($6.99/월로 둘 다 제공)

## 3. 타겟 유저

### 3.1 1차 타겟 (MVP 단계)

- **20~35세 유럽 K-콘텐츠 팬**

### 3.2 2차 확장 타겟

- 한국 교환학생/워홀 준비생
- K-팝 팬덤 (10대 후반~20대 초반)
- 한국 기업 취업 준비생
- 한국 여행 준비 관광객

## 4. 핵심 기능 — 광고 보상형 AI 설계

### 4.1 수익 모델의 핵심: Rewarded AI

**"AI 기능 = 비용 발생 = 광고로 상쇄"**라는 단순한 원칙.

| 사용자 유형 | AI 회화 접근 방식 | 비용 구조 |
|---|---|---|
| 무료 유저 | 리워드 광고 시청 후 AI 회화 5분 이용권 획득 | 광고 수익이 AI 비용 2~5배 커버 |
| 광고 제거 구매자 ($4.99 원타임) | 광고 없음, 일일 20분 AI 회화 | 낮은 사용량 → 비용 통제 가능 |
| Pro 구독자 ($7.99/월) | 무제한 AI 회화 + 프리미엄 시나리오 | 구독료가 AI 비용의 5배 이상 |

### 4.2 광고 보상 세부 설계

**기본 흐름**

1. 유저가 "회화 시작" 탭
2. "15초 광고 보고 5분 무료 회화 시작하기" 안내
3. 리워드 광고 시청 완료
4. AI 회화 타이머 5분 시작
5. 타이머 종료 시 "광고 한 번 더 보면 5분 추가!" 제안

**경제성 계산 (유럽 유저 기준)**

- 리워드 광고 1회 eCPM: **$12~18** (AdMob 유럽 리워드 평균)
- 광고 1회 시청 실현 수익: **약 $0.012~0.018**
- Claude Haiku 4.5 기준 5분 회화 API 비용: **약 $0.003~0.005**
- **마진: 광고 시청 1회당 $0.007~0.015 순이익**

즉, 광고를 많이 볼수록 앱이 돈을 버는 구조.

### 4.3 AI 회화 핵심 기능

**① 상황별 롤플레이 (Must-Have)**

초급/중급/고급 각 30개씩 총 90개 시나리오로 시작:

- 초급: 카페 주문, 자기소개, 길 묻기, 가격 묻기, 시간 약속
- 중급: 친구와 약속 잡기, 식당 추천 받기, 한국 문화 설명 듣기
- 고급: 면접 준비, 한국 뉴스 토론, K-드라마 장면 재연

**② 실시간 교정 피드백**

유저가 말한 문장에 대해 AI가 3단계 피드백 제공:

1. **문법 체크** — 틀린 조사·어미 강조
2. **자연스러움 점수** — 1~10점, 한국인이 실제로 쓰는 표현인가
3. **더 자연스러운 대안** — "이렇게 말하면 더 자연스러워요"

**③ 발음 피드백 (v1.1에서 추가 예정)**

v1.0에서는 텍스트 기반만 (STT 비용 절감). 반응 좋으면 음성 인식 추가.

**④ 복습 시스템**

- 대화에서 사용한 새 단어/표현 자동 저장
- 기존 단어 앱과 연동하여 단어장에 추가
- 주 1회 "이번 주 배운 표현" 퀴즈 제공

### 4.4 기존 단어 앱과의 연동

- 기존 앱에서 학습한 단어 리스트 API 연동
- AI 프롬프트에 "이 유저가 최근 학습한 단어: [리스트]" 주입
- AI가 유저 수준에 맞는 단어로 대화 유도

## 5. 기술 스택

### 5.1 프론트엔드

```
Expo SDK 52+ / React Native
TypeScript
Expo Router
NativeWind (Tailwind for RN)
Zustand (상태 관리)
```

### 5.2 AI 및 서버

```
Claude Haiku 4.5 (기본 회화) — $1/$5 per MTok
Claude Sonnet 4.6 (고급 교정/분석) — $3/$15 per MTok
Vercel Serverless Functions (API 프록시) — 무료 티어 시작
Supabase (유저 사용량 추적 DB) — 무료 티어 시작
```

**중요: API 키는 절대 클라이언트에 노출 금지.** Vercel Functions를 중간에 두어 유저별 rate limit 관리.

### 5.3 수익화 SDK

```
react-native-google-mobile-ads — 리워드 광고 + 배너
RevenueCat — 구독 관리
```

### 5.4 비용 최적화 전략

**프롬프트 캐싱 적극 활용**
- 시나리오 설명, 시스템 프롬프트를 캐싱 → 90% 비용 절감
- 유저별 학습 단어 리스트는 1시간 캐싱

**모델 라우팅**
- 일반 대화: Haiku 4.5 (저렴)
- 문법 교정/심화 피드백: Sonnet 4.6 (고품질)
- 예상 비율: Haiku 80% + Sonnet 20%

**컨텍스트 길이 제한**
- 한 대화 세션 최대 10턴으로 제한 → 컨텍스트 무한 증가 방지
- 10턴 넘으면 "새 대화 시작" 유도

## 6. 수익 모델 상세

### 6.1 3단계 수익 구조

**Tier 1: 무료 (광고 보상형)**
- 매일 무료 대화 시간 0분 (광고 시청해야 시간 획득)
- 리워드 광고 1회 = 5분 대화
- 일일 최대 30분까지 광고로 획득 가능
- 기본 시나리오 50개 제공

**Tier 2: Ad-Free ($4.99 원타임)**
- 모든 광고 제거
- 일일 20분 AI 회화 자동 지급
- 기본 시나리오 50개 + 중급 30개
- 복습 시스템 포함

**Tier 3: Pro 구독 ($7.99/월 또는 $59.99/년)**
- Ad-Free 포함
- 무제한 AI 회화
- 모든 시나리오 90개+ (고급 포함)
- 프리미엄 교정 (Sonnet 4.6)
- 주간 학습 리포트 이메일
- 기존 단어 앱과의 통합 번들

### 6.2 수익 예상 (MAU 3만 기준, 유럽 중심)

| 항목 | 전환율 | 유저/세션 | 월 매출 |
|---|---|---|---|
| 리워드 광고 (무료 유저) | 평균 10회/월 | 25,000명 × 10회 | $3,000 (eCPM $12) |
| 배너 광고 | 항시 노출 | 25,000명 | $500 |
| Ad-Free 원타임 ($4.99) | 2% | 600명 | $180 (평균화) |
| Pro 구독 ($7.99) | 3% | 900명 | $7,200 |
| **합계 (Gross)** | - | - | **약 $10,880** |

- 스토어 수수료 15% 제외: **약 $9,248**
- AI API 비용 (예상): **약 $800~1,500**
- **순이익: 약 $7,500~8,400/월**

기존 단어 앱의 유저 일부만 전환해도 빠르게 도달 가능한 수치.

### 6.3 핵심 방어 로직

**비용 폭증 방지 장치 필수**

- 유저별 일일 최대 토큰 사용량 제한 (Supabase로 추적)
- 비정상 사용 감지 시 자동 차단 (봇/어뷰저)
- 구독자도 일 100분 fair-use policy 적용
- API 호출당 최대 출력 토큰 300으로 제한

- 기존 단어 앱 코드베이스에서 신규 프로젝트 분기

- 채팅 UI (메신저 스타일)
- 시나리오 선택 화면
- 초급 시나리오 30개 프롬프트 작성
- 교정 피드백 UI (3단계 카드 디자인)

### Week 5-6: 수익화
- AdMob 리워드 광고 통합
- 광고 시청 → 시간 지급 로직
- RevenueCat 구독 플로우
- Ad-Free 원타임 결제

### Week 7: 통합 + 폴리싱
- 기존 단어 앱과 딥링크 연동
- 온보딩 플로우 (K-드라마 장면 예시로 후킹)

### Week 8: QA + 소프트 런치
- 독일/프랑스 소프트 런치
- 리뷰 수집 + 긴급 버그 수정
- 프리미엄 시나리오 추가 (v1.0 이후)

## 8. ASO 전략

### 8.1 앱 이름 (확정)

**공식 앱스토어 표기명**: `KoreanTalk: AI Conversation Practice`

**네이밍 전략 (B안 채택)**

브랜드(KoreanTalk) + 콜론 + 기능 설명(AI Conversation Practice) 구조. Duolingo, Headspace, ELSA Speak 등 글로벌 성공 앱들이 공통적으로 쓰는 검증된 패턴.

**이 구조의 3가지 이점**

1. **ASO 검색 유리** — 앱스토어 알고리즘은 앱 이름 + 서브타이틀 전체를 인덱싱하므로 "AI", "Korean", "conversation", "practice" 키워드 모두 검색 노출됨
2. **브랜드 자산 축적** — 메인 식별자가 "KoreanTalk" 한 단어라서 유저 사이에서 "KoreanTalk 써봐"로 자연스럽게 회자됨. 장기적으로 브랜드 가치 상승.
3. **확장성 확보** — 나중에 "KoreanTalk Lite", "KoreanTalk Kids", "KoreanTalk Business" 등 시리즈화 가능

**플랫폼별 표기**

| 요소 | 값 | 비고 |
|---|---|---|
| App Store 이름 | KoreanTalk | 메인 타이틀 (30자 제한) |
| App Store 서브타이틀 | AI Conversation Practice | 부제 (30자 제한) |
| Google Play 이름 | KoreanTalk: AI Conversation Practice | 전체 표기 (50자 제한) |
| 짧은 설명 (Play) | Speak Korean naturally with AI | 80자 내 |
| 도메인 | koreantalk.app 또는 koreantalk.io | 확보 필요 |
| 소셜 핸들 | @koreantalkapp | 통일 권장 |

**유저가 부르는 이름 (브랜드 일관성)**

모든 커뮤니케이션에서 `KoreanTalk` 한 단어로 통일. "AI KoreanTalk", "Korean Talk App" 같은 변형 표기 금지. 일관된 브랜드 노출이 핵심.

### 8.2 키워드 타겟

- 1순위: korean conversation, learn korean speaking, korean AI tutor
- 2순위: practice korean, korean fluency, korean language partner
- 3순위: topik speaking, k-drama korean, korean pronunciation

### 8.3 스크린샷 구성 (6장)

1. 시나리오 선택 — "Pick a real-life situation"
2. AI 대화 화면 — "Talk freely. Make mistakes. Get better."
3. 교정 피드백 — "Understand why, not just what"
4. 광고 리워드 — "Watch an ad, get 5 min of free practice"
5. Pro 혜택 — "Unlimited talk with premium tutor AI"
6. 기존 앱 연동 — "Your vocabulary, now in conversation"
