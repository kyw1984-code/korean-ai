❓ 자주 묻는 질문

  • 무료로 사용할 수 있나요?
    네, 기본 대화는 무료입니다. 광고 시청 시 추가 이용 시간을 얻을 수 있으며, 광고 없는 무제한 사용은 프리미엄 구독으로 이용 가능합니다.
  • 구독을 취소하려면 어떻게 하나요?
    iPhone의 설정 앱 → Apple ID → 구독 → KoreanTalk에서 취소할 수 있습니다.
  • 이전 구매를 복원하려면?
    앱 내 설정 화면 → "구매 복원" 버튼을 탭하세요.
  • 어떤 언어를 지원하나요?
    현재 한국어 학습만 지원합니다. 향후 다른 언어 지원을 검토 중입니다.
